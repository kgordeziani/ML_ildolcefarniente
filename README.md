# ML_ildolcefarniente
Automatic exercise recognition in video

## პროექტის მიმოხილვა
ჩვენი პროექტის მიზანია ვიდეოებიდან ფიზიკური ვარჯიშების ავტომატური ამოცნობა და კლასიფიკაცია სხვადასხვა მახასიათებელზე დაყრდნობით, კერძოდ, ადამიანის მოძრაობის დინამიკის, სხეულის საკვანძო წერტილებისა და კადრების დროითი თანამიმდევრობის გათვალისწინებით. სისტემამ უნდა შეძლოს ზუსტად განსაზღვროს, თუ რა კონკრეტულ ვარჯიშს (მაგალითად: ჩაჯდომას, აზიდვას, ორძელზე ვარჯიშს და ა.შ.) ასრულებს ადამიანი მოცემულ ვიდეოში.
ჩვენი მონაცემები შედგება ორი ძირითადი ნაწილისგან: Penn Action Dataset და Kaggle Workout Videos. პირველი დატასეტი შეიცავს 2,326 ვიდეოს, რომლებსაც უკვე მოჰყვება ადამიანის მიერ ხელით დამუშავებული 13 ძირითადი სახსრის კოორდინატები, ხოლო მეორე ნაწილი მოიცავს რეალურ გარემოში გადაღებულ ფიტნეს ვიდეოებს ყოველგვარი წინასწარი ანოტაციების გარეშე. ამ ამოცანასა და მოდელების ეფექტურობას შევაფასებთ რამდენიმე მეტრიკის — Accuracy-ს, F1-Score-ის, Training Time-ისა და რეალურ დროში მუშაობის სიჩქარის (Inference speed - ms/frame) გამოყენებით.

### მიდგომა ამ ამოცანის გადასაჭრელად
ამ ამოცანის გადასაჭრელად ეტაპობრივ მიდგომას ავირჩევთ და შევადარებთ ორ ძირითად, რადიკალურად განსხვავებულ არქიტექტურულ Pipeline-ს ავაგებთ: ორ პარალელურ pipeline-ს (პირველს ჩავშლით კიდევ ორ ნაწილად) შევადარებთ ერთმანეთს.

პირველი გზა ეფუძნება სხეულის Keypoints დინამიკასა და BiLSTM ქსელებს. ეს არის მსუბუქი მიდგომა (~83,000 პარამეტრი), რომელიც ორიენტირებულია რეალურ დროში მუშაობაზე (Real-time inference). ვიდეოებიდან MediaPipe BlazePose-ის საშუალებით ამოვიღებთ სხეულის "ჩონჩხს", ხოლო ფიჩერებად გამოვიყენებთ სახსრების კუთხეებსა და კოორდინატებს, რომლებსაც 30-კადრიანი მიმდევრობებით მივაწვდით BiLSTM კლასიფიკატორს. აქ გამოვიყნებთ ორ მიდგომას. პირველში ავიღებთ Penn Action-ის ორიგინალ 13 წერტილს, ხოლო Kaggle-ის მონაცემებიდანაც მხოლოდ ამ 13 შესაბამის წერტილს დავტოვებთ, რათა ორი დატასეტი საერთო ბაზაზე შევაერთოთ. მეორეგან კი Penn Action-ის ნედლ ვიდეოებსაც თავიდან გადავამუშავებთ MediaPipe-ით, ისევე როგორც Kaggle-ის ვიდეოებს, და მოდელს სრულ 33 საკვანძო წერტილზე გავწვრთნით. ეს საშუალებას მოგვცემს დავინახოთ, რამდენად ზრდის სიზუსტეს დამატებითი წერტილების (მაგ. სახისა და ტერფების დეტალების) გამოყენება.

მეორე Pipeline არის ვიდეოზე დაფუძნებული მიდგიმა. ეს არის "End-to-End" მიზანმიმართული ვიზუალური მიდგომა, სადაც ყოველგვარი Keypoint-ების გამოყოფის  გარეშე, პირდაპირ ნედლ ვიდეო კადრებზე (Raw video frames) დავატრეინიგბეთ. გამოვიყენებთ  Kinetics-400-ზე წინასწარ დასწავლილ ძლიერ 3D CNN მოდელებს (X3D ან SlowFast).
